I  - 	Autenticação no GIT HUB
	Para autenticação no GIT GUB é necessária a instalação de uma chave SSH
	no computador local e a vinculação desta mesma chave no GIT HUB.
	O link a seguir descreve os passos a serem realizados para concluir essa
	tarefa:
	https://cursos.alura.com.br/forum/topico-permission-denied-publickey-82429

	Complementarmente, talvez seja necessário ativar ssh-agent:
	https://bobbyhadz.com/blog/unable-to-start-ssh-agent-service-error-1058-powershell

	Se necessário utilizar o comando ssh-agent bash para adicionao o bash à lista de clientes do
	ssh.
